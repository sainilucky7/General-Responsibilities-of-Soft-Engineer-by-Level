# General-Responsibilities-of-Soft-Engineer-by-Level

# •	L1 Engineer (Entry-Level/Support):
o	Handles basic user queries, simple bugs, and scripted fixes. 
o	Often requires supervision and acts as the first point of contact. 
# •	L2 Engineer (Junior/Mid-Level):
o	Troubleshoots more complex issues needing deeper code/product understanding. 
o	Can manage end-to-end features and might be an intern or new grad. 
# •	L3 Engineer (Mid-Level/Senior):
o	Tackles core system bugs, crafts patches, and works more independently.
o	Designs smaller features, mentors juniors, and leads small projects. 
# •	L4 Engineer (Senior/Lead):
o	Designs system architecture, resolves structural issues, or leads teams.
o	Often has significant experience and can handle complex, well-defined projects.
